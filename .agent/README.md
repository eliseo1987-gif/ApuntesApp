# Guía de Interacción con el Agente (Antigravity) 🥷

Este directorio `.agent` contiene el cerebro y las herramientas que utilizo para ayudarte. He configurado flujos de trabajo específicos para que nuestra colaboración sea más eficiente.

## 🚀 Comandos de Flujo (Workflows)

Puedes pedirme que ejecute flujos de trabajo específicos usando comandos o simplemente describiendo la tarea. He configurado estos para ti:

- **`/analisis`**: Haré un chequeo completo del sistema, detectaré fallos de UX y te propondré mejoras de negocio.
- **`/redesenar`**: Optimizaré visualmente cualquier componente bajo la estética *Scientific Luxury*.
- **`/nueva-funcion`**: Seguiré un proceso estructurado desde el brainstorming hasta el despliegue para nuevas características.

## 🛠️ Superpoderes (Skills)

Tengo acceso a más de 30 habilidades especializadas, incluyendo:

- **`frontend-design`**: Para crear interfaces nivel premium.
- **`project-development`**: Para estructurar el crecimiento del ERP de forma escalable.
- **`ui-ux-pro-max`**: Inteligencia de diseño aplicada a cada pixel.

## 💡 Consejos para Mejorar nuestra Interacción

1. **Contexto es Rey**: Si mencionas un archivo específico con `@[archivo]`, podré analizarlo con prioridad.
2. **Pide el "Por qué"**: Si tomo una decisión técnica, no dudes en preguntarme la razón. Me baso en mejores prácticas de ingeniería.
3. **Iteración Rápida**: Prefiero construir algo funcional, mostrártelo y luego pulirlo (estilo Lean Startup).
