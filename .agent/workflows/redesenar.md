---
description: Rediseñar un componente específico con estética Premium/Scientific Luxury
---

Usa este workflow cuando el usuario pida mejorar visualmente una parte del sistema.

1. **Definición del Alcance**:
   - Preguntar al usuario qué componente desea rediseñar si no está claro.
   - Identificar el código actual del componente en `index.html` y sus estilos.

2. **Inspiración y Estilo**:
   - Usar la skill `frontend-design` para generar un concepto.
   - Aplicar principios de "Scientific Luxury": cristales, gradientes sutiles, tipografía moderna, micro-animaciones.

3. **Prototipado**:
   - Generar una imagen de referencia con `generate_image` si es necesario para validar con el usuario.
   - Proponer el nuevo código HTML/CSS.

4. **Aplicación**:
   - Reemplazar el código viejo con el nuevo, asegurando que no se pierda funcionalidad.
   - Verificar la responsividad.
